## Research & Data Collection

### Web Search Strategy

For each business context, conduct targeted searches:

```
Search Pattern:
1. "[industry] market size [country/region] [year]"
2. "[target customer] behavior trends [region]"
3. "[competitor name] pricing features review"
4. "[industry] regulations [country]"
5. "[industry] startup funding landscape"
6. "[customer segment] pain points survey"
```

### Data Sources

| Data Need | Source Type | Access Method |
|-----------|------------|---------------|
| Economic indicators | World Bank, national statistics | APIs, web search |
| Industry reports | Market research firms | Web search, purchase summaries |
| Competitor data | Public filings, websites | Web scraping, manual review |
| Demographics | Census data, surveys | Web search, data portals |
| Trends | Google Trends, social media | APIs, web search |

## Synthesis & Analysis

### Pattern Recognition Workflow

```
Raw Data → Categorization → Theme Extraction → Insight Formation → Validation

1. Categorization: Group data points by type (pain, gain, behavior, quote)
2. Theme Extraction: Identify recurring patterns across categories
3. Insight Formation: Convert themes into actionable insights
4. Validation: Cross-check with source data, check for bias
```

### Agent Capability Mapping

When executing, match available tools to these capabilities:
- **Web Search**: Market research, competitor analysis, trend identification
- **Data Sources**: Financial data, economic indicators, industry metrics
- **Code Execution**: Financial modeling, data analysis, visualization
- **Image Generation**: Persona photos, concept illustrations, diagrams
- **Document Creation**: Business plans, pitch decks, formal documents
- **Slide Creation**: Presentations, canvases, visual frameworks

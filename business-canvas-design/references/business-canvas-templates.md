### Lean Canvas - Detailed Guide

**PROBLEM**: List the top 1-3 problems your target customers face. Be specific and evidence-based. List existing alternatives they currently use.

**SOLUTION**: Top 1-3 features that solve the identified problems. Keep high-level; specific features come later.

**UNIQUE VALUE PROPOSITION**: A single, clear sentence that explains why you're different and worth buying. Answer: What do you do? Who do you serve? Why are you different?

**UNFAIR ADVANTAGE**: Something that cannot be easily copied or bought. Examples: insider knowledge, personal authority, community, existing customers, network effects, patents.

**CUSTOMER SEGMENTS**: Who are your target customers? Be specific. Identify early adopters who feel the pain most acutely.

**KEY METRICS**: The 3-5 metrics that matter most for your business stage. Common: activation rate, retention rate, revenue, NPS, CAC, LTV.

**COST STRUCTURE**: Fixed costs (team, office, infrastructure) vs. variable costs (marketing, COGS, customer support).

**REVENUE STREAMS**: How you make money. Pricing model, estimated LTV, payment terms.

## Value Proposition Canvas

Best for: Deep customer-value alignment, product-market fit analysis

```
┌────────────────────────────────────────┬─────────────────────────────────────────┐
│           CUSTOMER PROFILE             │           VALUE MAP                     │
│                                        │                                         │
│  GAINS 收获                             │  GAIN CREATORS 收获创造                  │
│  • Desired outcomes                    │  • How we create gains                  │
│  • Benefits sought                     │  • Value-adding features                │
│  • Aspirations                         │  • Unexpected delights                  │
│                                        │                                         │
│  PAINS 痛点                             │  PAIN RELIEVERS 痛点缓解                │
│  • Problems encountered                │  • How we eliminate pains               │
│  • Risks, obstacles                    │  • Frustration removers                 │
│  • Negative emotions                   │  • Risk reducers                        │
│                                        │                                         │
│  CUSTOMER JOBS 客户工作                 │  PRODUCTS & SERVICES                    │
│  • Functional tasks                    │  • What we offer                        │
│  • Emotional needs                     │  • Core features                        │
│  • Social obligations                  │  • Key capabilities                     │
│                                        │                                         │
│  [Priority ranking]                    │  [Alignment check]                      │
└────────────────────────────────────────┴─────────────────────────────────────────┘

FIT CHECK: Does each high-priority customer job, pain, and gain have a corresponding
solution element? Are we over-engineering low-priority items?
```

## SWOT Analysis

Best for: Strategic planning, identifying risks and opportunities

```
┌──────────────────────────────┬──────────────────────────────┐
│      STRENGTHS 优势           │      WEAKNESSES 劣势         │
│      (Internal/Positive)     │      (Internal/Negative)     │
│                              │                              │
│ • What do we do well?        │ • What do we lack?           │
│ • What resources do we have? │ • What do others do better?  │
│ • What is our edge?          │ • Where are we vulnerable?   │
│ • What do customers value?   │ • What limits our growth?    │
│                              │                              │
├──────────────────────────────┼──────────────────────────────┤
│      OPPORTUNITIES 机会       │      THREATS 威胁            │
│      (External/Positive)     │      (External/Negative)     │
│                              │                              │
│ • Market trends?             │ • Competitor actions?        │
│ • Technology changes?        │ • Regulatory changes?        │
│ • Customer shifts?           │ • Economic conditions?       │
│ • Partnership possibilities? │ • Market saturation?         │
│                              │                              │
└──────────────────────────────┴──────────────────────────────┘

STRATEGIC IMPLICATIONS:
S+O: Leverage strengths to capture opportunities
S+T: Use strengths to defend against threats
W+O: Address weaknesses to pursue opportunities
W+T: Defensive strategy; minimize risks
```

## Canvas Selection Guide

### By Business Maturity

| Stage | Primary Canvas | Secondary Canvas |
|-------|---------------|-----------------|
| Idea validation | Lean Canvas | Value Proposition Canvas |
| Problem-solution fit | Value Proposition Canvas | Empathy Map + POV |
| MVP planning | Lean Canvas + GTM Canvas | Business Model Canvas |
| Product-market fit | Business Model Canvas | Competitive Analysis |
| Scaling | Business Model Canvas + Financial Projection | SWOT + GTM Canvas |

### By Output Goal

| Goal | Recommended Canvas |
|------|-------------------|
| Investor pitch | Lean Canvas + Financial Projection |
| Internal alignment | Business Model Canvas |
| Product direction | Value Proposition Canvas |
| Launch planning | GTM Canvas + Competitive Analysis |
| Strategic review | SWOT + Business Model Canvas |
| Fundraising | Lean Canvas + Financial Projection + Competitive Analysis |
